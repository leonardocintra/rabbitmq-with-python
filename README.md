# RabbitMQ with Python
Simples sistema de fila, senha, etc

# Obs
Subir o docker primeiro
- Postgress
- RabbitMQ

```
sudo docker-compose up -d
```

# How to use
Voce inicia os dois arquivos `receive.py` e `send.py` em terminais diferentes

Term1
```
python3 receive.py
```

Term2
```
python3 send.py
```

Boa brincadeira!
